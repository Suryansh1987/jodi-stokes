## Proposed Replacement

- Prefer 21st.dev `Large Name Footer` as the primary replacement direction.
- Preserve current brand copy, link groups, social links, legal text, and placeholder comments.
- Use Magic UI `dot-pattern` only as a subtle background primitive if it improves depth without reducing readability.

## Why It Was Chosen

21st.dev `Large Name Footer` is a full footer component and better matches a brand-forward lifestyle site than a background texture alone.

## Estimated Implementation Complexity

Medium

## Dependencies

- 21st.dev `Large Name Footer`
- Existing `next/link`
- Existing shadcn `Button` if the selected footer includes calls to action
- Custom brand/social icons may need to replace sample icons
- Optional Magic UI `dot-pattern`

## Are Motion Animations Alone Sufficient?

No. This task does not need motion; static texture is the safer enrichment.

## Acceptance Notes

- Footer links and placeholders must remain intact.
- Do not duplicate the topbar marquee in the footer.
- Texture must not interfere with focus-visible states.
